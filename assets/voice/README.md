# Voice Assets

This folder contains audio files for the fake call feature.

## Required Files:
- `ringtone.mp3` - The ringtone sound that plays when a fake call comes in
- `voice.mp3` - Default voice file (optional, users can upload custom voices)

## Adding Ringtone:
1. Find or create a ringtone audio file (MP3 format recommended)
2. Name it `ringtone.mp3`
3. Place it in this folder

You can download free ringtones from:
- https://www.zedge.net/ringtones
- https://www.freesound.org/

Or use any MP3 audio file you want as a ringtone.
